## 起動方法
まずGirakbackendを起動させます
方法は→
https://github.com/girakdev/girack-backend

起動させたら
ファイルのディレクトリに行って
```
yarn install
```
これで必要なファイルのダウンロードが始まるはずです。(package.jsonを元に)

次に以下のコマンドでビルドして
```
yarn build
```
それで以下のコマンドで起動できます
標準ポートは3000となっています。
```
yarn start
```
