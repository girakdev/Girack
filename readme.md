## 始め方
ファイルのディレクトリに行ってこれ↓
```
yarn install
```
これで必要なファイルのダウンロードが始まるはずです。(package.jsonを元に)

んでこれでサーバーそのものをスタート。
標準ポートは3000となっています。
```
node server.js
```
